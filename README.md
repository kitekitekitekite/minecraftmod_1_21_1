         **Star Wars Mod**

Minecraft Forge mod final java project I made for Minecraft 1.21.1 that adds Star Wars themed content.

 **Features**
____________________________________________________________________________

Lightsabers: craftable Red and blue lightsaber items

 - Custom sounds for ignite, swing, and closing of the lightsabers

 - Better damage than netherite swords

 - Shown in the combat tab of the creative inventory

____________________________________________________________________________
 
 
 - Blocks: Custom blocks created for Star Wars themed builds

____________________________________________________________________________

 - Cantina Music Disc

  - New music disc that plays "Cantina"

   - Works with jukeboxes

_____________________________________________________________________________

 - Sith-Lord Zombie Reskin

  - All zombies reskinned with a custom "Darth Grimon" texture file

   - Default zombie behavior preserved

    - Optional: can be summoned holding a lightsaber
